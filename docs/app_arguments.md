# コマンドライン引数

## 共通オプション

|          オプション          |      内容      |   省略時   |                 備考                  |
| ---------------------------- | -------------- | ---------- | ------------------------------------- |
| -h<br>--help                 | ヘルプ表示     |            |                                       |
| -c CONFIG<br>--config CONFIG | 設定ファイル   | config.ini | 相対パス指定時は`app.py`から          |
| --service                    | 連携先サービス | slack      | slack<br>standard_io,std<br>web,flask |

## ロギングオプション

|  オプション   |                                             内容                                              | 省略時 | 備考 |
| ------------- | --------------------------------------------------------------------------------------------- | ------ | ---- |
| -d, --debug   | デバッグレベル<br>1: debug<br>2: trace                                                        | 0      |      |
| -v, --verbose | 動作ログ出力切替<br>1 (0x01): パラメータ&クエリ<br>2 (0x02): クエリ実行結果<br>3 (0x03): 両方 | 0      |      |
| --moderate    | ログレベルがエラー以下のもを非表示                                                            | False  |      |
| --notime      | ログフォーマットから日時を削除                                                                | False  |      |

## 個別オプション(standard_io専用)

| オプション |           内容           | 省略時 |            備考            |
| ---------- | ------------------------ | ------ | -------------------------- |
| --text     | 呼び出しキーワードを指定 |        | 空白区切りで引数も指定可能 |

## 個別オプション(web専用)

| オプション |       内容       |  省略時   | 備考 |
| ---------- | ---------------- | --------- | ---- |
| --host     | 起動するアドレス | 127.0.0.1 |      |
| --port     | 起動するポート   | 8000      |      |
