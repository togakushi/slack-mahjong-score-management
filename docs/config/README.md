# 設定ファイル
## 設定ファイルの種類
アプリケーションの設定はメイン設定(必須)とルールセット設定(オプション)から構成される。\
ルールセット設定はデフォルトのルールセットが準備されており、設定省略時はデフォルトルールセットが利用されるためオプション扱いとする。

メイン設定内にサービス個別の設定(slackやdiscordなど)が含まれる。\
サービス個別設定でチャンネル個別設定が定義されている場合は追加の設定ファイルが読み込まれる。
